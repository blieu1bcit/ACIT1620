About C and C++

Unlike python, which is an interepreted language, C and C++ are complied languages, which means the code you write in either language are not yet rendered by your computer to create something tangible. The code needs to be put into a compiler first, which will turn it into machine language, which your computer can finally read and turn it into actual content.
